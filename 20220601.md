- git 
  - git clone [클론할 레포 주소]
  
    (클론하는 방법)
  
  - git remote add origin [주소]
  
    (오리진이라는 이름으로 리모트 연결)
  
  - git add .
  
    (현재 브랜치에 있는 거 다 에드(gitignore에 있는거 빼고))
  
  - git commit -m "커밋 메모"
  
    (커밋하는 범)
  
  - git push origin main
  
    (로컬의 메인하고 리모트의 메인하고 같도록 만듦)
  
  - git pull origin main
  
    (리모트에 있는 메인하고 로컬에 있는 메인하고 같도록 만듦)
  
  - git branch
  
    (현재 브랜치 보여줌)
  
  - git status
  
    (현재 상태 확인)
  
  - git init 
  
    (이건 현재 폴더를 로컬 저장소로 만드는 것)
